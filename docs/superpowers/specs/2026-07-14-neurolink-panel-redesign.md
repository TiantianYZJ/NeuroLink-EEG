# NeuroLink 面板重设计规范

## 设计语言
遵循 DESIGN.md（Revolut 暗色系统）：
- 背景 `#000000`，卡片 `#16181a`，品牌色 `#494fdf`
- 药丸按钮 `border-radius: 9999px`，卡片圆角 `20px`
- 正文 Inter 400，标题大写 `letter-spacing: 1.2px`
- 分隔线 `rgba(255,255,255,0.06)`

## 布局（桌面端）

### Header
```
← 返回 面板名称 | 4.0s | 12,847帧 | 02:14 | ● 已连接 | 120 Hz
```
- 状态色点（绿/红）表示连接
- 采样率钴紫药丸边框
- 系统数据（窗口/帧数/时长）用竖线分隔

### 主内容 Grid
两列：左 1.5fr / 右 1fr

**左列：**
1. 实验控制卡片 — 倒计时 52px 粗体 + 剩余/总计双行 + 进度条 + 全阶段条 + 按钮行 + 自动模式
2. 频带功率/实时指标 — 左右并排

**右列：**
1. 脑区热力图 — 200×200 canvas + 色阶 + 恢复状态行（压缩）

**全宽：**
1. 实时脑电波 — 跨两列 canvas + 采样率选择器 + 原始数据框

### 快捷键
- Space: 暂停/继续
- Enter: 下一阶段
- Z/X/C/V: 打标

## 响应式
≤900px 时切换到单列竖排。
