### 一、研究核心概要
- **研究目的**：量化跨学科任务切换对心流状态的破坏程度，并基于便携式EEG头环精准测算心流被打断后的脑电恢复时长。
- **核心范式**：构建“稳态心流诱发 → 跨学科切换干预 → 状态恢复观测”的递进式实验。
- **核心对比**：同学科连续任务（对照组） vs. 跨学科切换任务（实验组）。

### 二、研究问题与假设
| 维度 | 内容 |
| :--- | :--- |
| **核心问题** | ①跨学科切换是否显著破坏心流稳态？②不同切换类型（文理/理艺/文艺）破坏程度是否不同？③EEG指标恢复需多久？④主观体验与客观指标是否相关？ |
| **研究假设** | ①跨学科切换后心流EEG特征显著回落，差于同学科条件；②思维范式差异越大（如数理→艺术），破坏越大、恢复越慢；③主观心流评分与专注节律正相关，与恢复时长负相关。 |

### 三、实验变量与量化指标体系
- **自变量**：任务切换类型（同学科 vs. 跨学科）；跨学科维度（文理切换、理艺切换、文艺切换）。
- **因变量**：心流EEG稳态指标、状态衰减幅度、EEG恢复时间、认知转换损耗。
- **控制变量**：任务难度/时长、环境（噪音/光线）、被试状态（情绪/作息/熟练度）。

**核心EEG量化指标（适配前额叶便携式头环）：**
| 类别 | 具体指标 | 含义 |
| :--- | :--- | :--- |
| **心流核心指标** | Theta/Alpha比值、Alpha波能量、Beta波能量 | 比值稳定为心流稳态；Alpha反映沉浸放松，Beta反映主动专注。 |
| **认知损耗指标** | Gamma波能量、脑电熵值、认知负载指数 | 切换后数值升高，代表认知损耗增大、心流被破坏。 |
| **恢复时间标准** | 连续30s回归至初始心流稳态均值±5%区间 | 从切换结束到此达标的时间差，定义为EEG恢复时长。 |

- **主观指标**：采用简化心流体验量表（FSS），测评专注度、忘我感、时间扭曲、投入度、流畅度，用于交叉验证。

### 四、被试与设备
- **被试**：4名健康学生/青年（16-24岁），无精神/睡眠/脑外伤史，实验前24h无熬夜/咖啡因/剧烈运动。
- **设备**：便携式干电极EEG头环（前额叶，≥250Hz，无线传输）；静音恒温环境；标准化任务终端与电子问卷。

### 五、实验任务设计（三类基础学科任务）
| 任务类型 | 内容 | 认知模式 |
| :--- | :--- | :--- |
| **A类（数理逻辑）** | 推理、逻辑运算 | 理性分析、高认知控制、低感性 |
| **B类（语言人文）** | 文本精读、赏析、短文创作 | 语义理解、情感感知、高感性 |
| **C类（艺术创想）** | 构图、色彩搭配、创意构思 | 直觉思维、发散创造、低规则约束 |

**实验分组**：
- **对照组**：单一任务持续（如A→A→A），无切换。
- **实验组**：三类跨范式切换（A→B文理切换、A→C理艺切换、B→C文艺切换）。

### 六、完整实验流程（单次约40分钟，被试内设计，顺序随机）
| 阶段 | 时长 | 操作要点 |
| :--- | :--- | :--- |
| **1. 前期准备** | 5min | 佩戴头环，采集1min静息态基线，讲解流程，平复情绪。 |
| **2. 稳态心流诱发** | 8min | 执行单一基础任务，取后4分钟稳定数据作为**个体心流基准稳态数据**。 |
| **3. 任务切换干预** | 2min | 对照组继续原任务；实验组无缝切换至跨学科任务，记录切换瞬间波动。 |
| **4. 心流恢复观测** | 10min | 回归初始原学科任务，实时追踪EEG指标，判定恢复至稳态的时间节点。 |
| **5. 数据采集与问卷** | （组间休息3min） | 每组结束后填写心流量表，同步保存EEG、行为数据。 |

### 七、数据处理与分析方法
- **预处理**：剔除干扰数据，划分**基线期、心流稳态期、切换干扰期、恢复期**四个阶段。
- **分析维度**：
  1. **组间差异**（t检验）：对比同学科 vs. 跨学科的衰减幅度、负载峰值、恢复时长。
  2. **切换类型差异**（单因素方差分析）：对比文理/理艺/文艺三类切换的差异。
  3. **相关性**（皮尔逊相关）：主观评分与客观EEG、恢复时间的关系。
  4. **时序特征**：绘制动态曲线，展示“稳态—跌落—恢复”全过程。

### 八、实验创新点
1. **场景创新**：聚焦真实高频的跨学科切换，突破单一任务局限。
2. **量化创新**：首次用便携式EEG精准量化心流衰减幅度与客观恢复时长。
3. **范式创新**：构建“诱发—切换—恢复”完整范式，区分不同切换类型的损耗差异。
4. **设备创新**：便携式头环，操作简便、可重复性高，易推广。

### 九、研究意义
- **理论**：填补跨学科切换与心流神经机制的研究空白，丰富心流动态变化理论。
- **实践**：为课业安排、学习节奏规划提供脑科学依据，指导规避无效切换、提升沉浸式学习效率。

### 十、实验控制与误差规避措施
- **环境**：统一静音、恒温、无视觉干扰。
- **顺序**：随机化实验条件，消除练习/疲劳效应。
- **难度**：预实验校准，确保各任务心流诱发效果一致。
- **状态**：筛查被试身心状态，禁止疲劳/焦虑/亢奋参与。
- **数据**：统一降噪筛选标准，人工核对时序节点，保证恢复时间量化精准。
