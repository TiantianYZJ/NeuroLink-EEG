# OpenBCI GUI 分析软件使用指南 - V1.1.0

## 一、 软件概况与基础操作

### 1. 软件简介
OpenBCI GUI 是用于可视化、记录和流式传输 OpenBCI 主板数据的软件工具。支持实时数据显示、数据回放，能以 CSV 或 BDF+ 格式保存数据，并支持将数据实时流式传输至第三方软件（如 MATLAB）。

### 2. 软件使用介绍
*   **下载地址**：https://openbci-cn.com/downloads
*   **启动方式**：
    *   Windows：解压后双击 `OpenBCI_GUI.exe`。
    *   macOS：双击 `OpenBCI_GUI`，如遇安全提示点击“打开”即可。
    *   Linux：终端进入目录，执行命令 `./OpenBCI_GUI`。
*   **设备连接**：参考配套的硬件使用指南连接传感板。
*   **数据存储**：默认保存在 `[USER]/Documents/OpenBCI_GUI/` 目录下，文件按生成时间自动命名。

## 二、 核心分析模块详解

### 1. 时间序列模块（Time Series）
*   **功能**：实时显示生物传感数据的主窗口，波形代表各通道检测到的电压（单位：µVrms）。
*   **通道控制**：各通道前方的数字序号即为通道开关（点击切换，灰黑色为关闭），建议关闭未使用的通道以防干扰。
*   **颜色与状态**：通道边框绿色代表有数据流，红色代表无数据流；底色蓝色代表该模块将使用该通道数据，灰色代表不使用。

### 2. 硬件设置（Hardware Settings）
*   **适用**：仅适用于 Cyton 8通道 或 Cyton+Daisy 16通道 传感板。
*   **前提**：需停止数据流才能访问设置界面。
*   **可调参数**：
    *   **PGA增益**：提供 x1, x2, x4, x6, x8, x12, x24 七个选项，用于放大微弱生物电信号、改善信噪比。
    *   **输入类型**：默认设置为“Normal”，用于记录生物信号。
    *   **偏置包含**（Bias Include）：开启可施加共模偏置，稳定信号并减少干扰。
    *   **SRB1**：全局设置。开启后，所有通道的负输入端会内部短接连接到一个公共参考电极（如耳夹）。
    *   **SRB2**：通道级设置。开启时，该通道负输入参考 SRB2 电极；关闭时，为差分测量（参考自身正输入）。允许在同一芯片上混用差分和参考测量。

### 3. 滤波器设置（Filters）
*   **Filter**：下拉选择“带通”或“带阻”滤波器。
*   **Notch（陷波）**：用于滤除 50Hz 或 60Hz 的工频交流电噪声。
*   **参数设置**：可分别设置各通道的起始/停止频率（Hz）、滤波器类型（如Butterworth）和阶数。支持保存、加载和重置过滤器配置。

### 4. 快速傅里叶变换图模块（FFT Plot）
*   **功能**：将时域信号转换为频域信号，显示不同频率上的振幅，帮助观察脑波频率分布。
*   **注意**：更改此模块的平滑度或滤波器设置，会同时影响“频带功率模块”的相应设置。

### 5. 加速度计模块（Accelerometer）
*   **功能**：显示传感板内置三轴加速度计的 X、Y、Z 当前加速度数值，以及三轴随时间变化的曲线图。

### 6. 头部拓扑图模块（Head Plot）
*   **功能**：在头部模型上以热力图形式显示大脑活动分布，红色越深表示活动越强烈。
*   **配置**：电极位置基于 10-20 系统标准，可自定义拖动。极性模式可切换显示“真实值（含正负极）”或“绝对值”。

### 7. 频谱图模块（Spectrogram）
*   **功能**：双谱图显示界面，观察 FFT 数据随时间的变化趋势。
*   **参数**：支持选择最大频率（Max Freq）、采样刷新率（Samples）、颜色渐变模式（Log/Lin）。
*   **重要提示**：此模块数据为近似值，仅供研究参考，不可用于临床诊断。

### 8. 专注度模块（Focus Widget）
*   **功能**：利用 BrainFlow Metric 算法检测放松或专注状态。
*   **参数设置**：
    *   **Channels（通道选择）**：选择参与计算的电极。
    *   **Metric（指标）**：选择 `Relaxation`（放松，闭眼冥想）或 `Concentration`（专注，睁眼集中）。
    *   **Classifier（分类器）**：支持 Regression（默认）、KNN、SVM 或 LDA。
    *   **Threshold（阈值）**：判断是否处于目标状态的临界值（0 到 1 之间）。
*   **听觉反馈**：点击 `Turn Audio On` 可开启声音反馈。支持 `Metric Value` 模式（音量随指标整体变化）和 `Band Powers` 模式（不同音调对应不同频率的脑波）。

### 9. 频带功率模块（Band Power）
*   **功能**：以柱状图直观展示五种主要脑电波的相对电压值。
*   **脑电波定义**：
    *   **Gamma (γ)**：32-100Hz，感知增强、解决问题、警觉状态。
    *   **Beta (β)**：13-32Hz，正常清醒意识、活跃思维、专注工作。
    *   **Alpha (α)**：8-13Hz，身体和精神放松。
    *   **Theta (θ)**：4-8Hz，深度放松、创造力、冥想。
    *   **Delta (δ)**：0.5-4Hz，深层无梦睡眠。

### 10. EMG 肌电模块
*   **功能**：将肌肉屈曲程度映射到 0-1 的数值，并可视化显示。
*   **核心逻辑**：通过动态阈值范围（上下阈值环）和自适应机制，将当前 EMG 平滑值映射到“舒适屈曲范围”。
*   **可调参数**：平滑度（影响响应速度）、μV极限（消除突发干扰）、蠕变速度（阈值环靠近速度）、最小 ΔμV（上下阈值的最小间隔）。

### 11. EMG 摇杆（EMG Joystick）
*   **功能**：利用最多4个通道的 EMG 信号控制一个摇杆，每个通道可映射到一个轴向方向（+X, -X, +Y, -Y）。
*   **操作**：顶部下拉菜单可更改控制映射关系，点击 `EMG Settings` 可微调摇杆灵敏度。

### 12. 网络模块（Networking）
*   **功能**：将采集的数据流传输至其他应用，支持传输数据类型：时间序列、FFT、EMG、频带功率、加速度计、脉搏、模拟/数字读数。
*   **支持的协议**：
    *   **Serial**：连续数据包发送，需匹配波特率和端口。注意：不推荐使用 Arduino 处理原始时间序列数据，建议使用频带功率数据。
    *   **UDP**：低延迟、容错，可同时向三个不同应用发送数据集。
    *   **LSL**：用于同步流数据，支持 MATLAB 和 Python（需安装 `pylsl` 库）。
    *   **OSC**：用于连接声音合成器、多媒体设备（如 MaxMSP, PureData）。

### 13. 标记模块（Marker）
*   **功能**：在数据流中插入事件标记，便于后续分析。
*   **添加标记的三种方法**：
    *   **界面按钮**：点击模块内的按钮。
    *   **键盘快捷键**：预设按键对应编号（Z=1, X=2, C=3, V=4, Shift+Z=5, Shift+X=6, Shift+C=7, Shift+V=8）。
    *   **UDP 输入**：监听指定 IP 和端口（可自定义）发送的浮点数值，记录为标记。
*   **数据保存**：标记会实时显示为垂直线，并保存到录制的 CSV 文件中（有专门的“标记通道”列）。

### 14. 信号质量检查（Cyton Signal / Ganglion Signal）
*   **Cyton 8/16 通道适用**：
    *   **阻抗模式**：注入电流检查电极接触状态。建议耐心等待，使用 `Check All Channels` 自动检查。此模式下数据不存入主文件。
    *   **实时模式**：通过时间序列图中“Railed/Not Railed”状态值判断。`Railed` 意味着信号超出发大器范围，通常是电极未接触皮肤或增益过高。
*   **Ganglion 4 通道适用**：注入电流检查，指示灯圆圈显示为绿色或黄色表示可接受的阻抗值。

### 15. 数据回放模块（Playback）
*   **功能**：在回放模式下，可选择 OpenBCI 的 `.txt` 回放文件加载数据进行离线分析。

### 16. 心率模块（Pulse）
*   **适用**：Cyton 处于实时模式时可用。
*   **功能**：从 Cyton 的 D11 引脚读取数据，计算每分钟心跳次数（BPM）和心跳间隔（IBI）。
